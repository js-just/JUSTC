# JUSTC [![Compile](https://github.com/js-just/JUSTC/actions/workflows/compile.yml/badge.svg)](https://github.com/js-just/JUSTC/actions/workflows/compile.yml)
JUSTC (Just an Ultimate Site Tool Configuration language) is a powerful, small, safe, easy-to-use object notation (or configuration) language designed to replace JSON (JavaScript Object Notation).

# Installation
```
npm i justc
```
